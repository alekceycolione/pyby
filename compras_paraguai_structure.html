O site usa uma estrutura de divs para listar os produtos. Cada produto está contido em um `div` com a classe `item-prod`. Dentro de cada `div` de produto, as informações são as seguintes: - **Nome do produto:** dentro de um `h2` com a classe `nome-produto` - **Preço:** dentro de um `div` com a classe `price-dolar` para o preço em dólar e `price-reais` para o preço em reais. - **Imagem:** a imagem principal está dentro de uma tag `img` com a classe `img-responsive`. - **Link para o produto:** O link para a página do produto está no `a` que envolve a imagem e o nome do produto.
